# Codes College Website 🎓

A modern, professional, and fully responsive educational website designed for **Codes College**. This project demonstrates industry-standard front-end development practices, including semantic HTML, advanced CSS3 techniques, and premium UI/UX design.

---

## 🌟 Key Features

- **Premium UI/UX:** Clean, modern design with a refined color palette (Deep Blue & Gold).
- **Fully Responsive:** Optimized for all screen sizes—from high-resolution desktops to mobile phones.
- **Glassmorphism Design:** Implementation of modern translucent backgrounds with backdrop filters.
- **Smooth Animations:** Integrated CSS keyframe animations for headers, sections, and interactive elements.
- **Semantic HTML5:** Built using SEO-friendly tags like `<header>`, `<nav>`, `<main>`, and `<footer>`.
- **Interactive Forms:** A fully styled Admission Form with real-time focus effects and validation-ready layout.
- **Grid & Flexbox Layouts:** Modern CSS techniques used instead of legacy table-based structures.

---

## 🛠️ Technologies Used

- **HTML5:** For structured and semantic content.
- **CSS3:** For advanced styling, animations, and responsive layouts.
- **Google Fonts:** Using the 'Outfit' typeface for a premium digital feel.
- **Matter.js & GSAP (Planned/Reference):** Foundation laid for future advanced interactive components.
